Para compilar em unix basta executar o makefile.
O objetivo deste trabalho foi implementar as estruturas de dados adequadas para simplificar a "expansão" do projeto na altura da segunda entrega.

Objetivos cumpridos:
Ler e guardar a informação de linhas e condutores armazenada em ficheiros.
Gerir as linhas e condutores.
Gerar e visualizar de modo (semi)formatado tabelas com horarios de uma paragem
Gerar e visualizar de modo formatado o horario de uma linha.
Inquirir sobe quais as linhas que incluem determinada paragem
Listar condutores disponiveis
Calcular e visualizar um percurso e tempos entre duas quaisquer paragens indicadas pelo utilizador

Não cumpridos:
Calcular, para uma linha especificada, quantos condutores são necessários (assumindo turnos com um número de horas fixo).
Visualizar o trabalho atribuído a um condutor.
